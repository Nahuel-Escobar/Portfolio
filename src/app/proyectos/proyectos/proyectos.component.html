3. Proyectos

Sistema de ventas

Sistema de ventas con 3 usuarios que tienen un rol cada uno, los cuales son: Administrador, Gerente, Empleado. Cada uno tendra diferentes permisos, se podra realizar una venta y se devolvera un ticket, la cual queda guardada en la base de datos y se puede acceder a este desde un historial de ventas.

Web Api Libros

Una Web de libros, es un crud donde se podran realizar todos los metodos http. Crear nuevos libros, Editarlos, Borrarlos y poder acceder a la información del libro. En Swagger se devuelven los status code correspondiente.

Sistema de historial clinico

Sistema Clinico donde se vera el historial clinico de un paciente, doctores, se veran las citas pendientes, se podra acceder al mantenimiento del paciente. Se otorgan permisos según el usuario, el cual dependiendo el usuario podra generar una nueva cita, ver datos del paciente y modificarlos.
